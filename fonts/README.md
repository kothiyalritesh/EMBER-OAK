# Custom Brand Fonts — EMBER & OAK

Place your licensed font files in this directory with these exact filenames:

## Required Files

| Font | Files Needed |
|------|-------------|
| MOURGEN | `MOURGEN.woff2`, `MOURGEN.woff`, `MOURGEN.ttf` |
| Queensides | `Queensides.woff2`, `Queensides.woff`, `Queensides.ttf` |
| Queensides Bold | `Queensides-Bold.woff2`, `Queensides-Bold.woff`, `Queensides-Bold.ttf` |
| Hongaria | `Hongaria.woff2`, `Hongaria.woff`, `Hongaria.ttf` |
| Porto Qastelo | `PortoQastelo.woff2`, `PortoQastelo.woff`, `PortoQastelo.ttf` |

## Font Usage in the Brand

| Font | Role | Used In |
|------|------|---------|
| **MOURGEN** | Brand wordmark / logo | Navbar logo, Loader, Footer |
| **Queensides** | Hero & section headings | Hero h1, Origin h2, Tasting h2, Nav links |
| **Hongaria** | Decorative labels & accents | All section eyebrow labels (e.g. "OUR RITUAL", "ORIGIN") |
| **Porto Qastelo** | Card titles & sub-headings | Menu card names, Ritual card titles, Feature titles, Author names |

## Converting Fonts

If you only have `.ttf` or `.otf` files, convert to `.woff2` for best performance:
- Use [Transfonter](https://transfonter.org/) (free, online)
- Use [Font Squirrel Webfont Generator](https://www.fontsquirrel.com/tools/webfont-generator)

## Fallback

If font files are missing, the site gracefully falls back to `Playfair Display` (loaded via Google Fonts) and then `Georgia`.
